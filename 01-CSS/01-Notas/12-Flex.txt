siempre se necesita un contenedor principal para poder utilizar flex

.container{
    border: 0.3rem solid black;
    display: flex;
    flex-direction: column;
    flex-wrap: wrap; -> si no tiene espacio se va colocando por abajo
}


.container{
    border: 0.3rem solid black;
    display: flex;
    justify-content: center; -> centra el contenido
}

justify-content: flex-end;-> todo a la izquierda o abajo

justify-content: space-around; -> se acomodan
justify-content: space-evenly; -> se acomodan a la misma medida

se alinean de forma vertical en el flexbox,tengo la sospecha de que si colocas
una medida como height, los cambio de flex se veran afectados en la misma 
direction

align-items: stretch; -> se estiran si no coloco una medida determinada y
se estira siempre al 100% del contenedor padre

align-items: baseline; -> lo mismo que el anterior, pero, tomará el espacio del
contenido, como que se apachurra

order : (numero) -> cambian las posiciones
